◆ ctr_led_value

typedef uint64_t ctr_led_value

Value for an LED.

Each consecutive byte corresponds to one channel of the underlying LED, starting with the least significant byte. If an LED doesn't have brigtness functionality, 255 is still used as the value for 'ON'.
